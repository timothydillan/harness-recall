## Task 2: Cursor Parser

**Files:** `src/harness_recall/parsers/cursor.py`, `tests/test_cursor_parser.py`, `tests/fixtures/cursor_state.vscdb`

Cursor stores chat data in SQLite at:
- `~/Library/Application Support/Cursor/User/globalStorage/state.vscdb`
- Table: `cursorDiskKV` (key TEXT, value BLOB)
- Keys: `composerData:{id}` (conversation metadata), `bubbleId:{composerId}:{bubbleId}` (messages)

Parser approach:
1. `discover()` finds state.vscdb files
2. `parse()` takes a conversation ID, reads composerData + all bubbles
3. New `parse_all()` method yields multiple Sessions from one DB file
4. Map: type=1 → user, type=2 → assistant
5. Tool calls from `toolFormerData` field
6. Timestamps from `createdAt` (ISO8601 or Unix ms)

Register as source "cursor" with default path `~/Library/Application Support/Cursor/User/globalStorage/`

Update parser registry, config defaults, CLI auto-index, README.

## Task 3: hrc browse TUI

**Files:** `src/harness_recall/tui.py`, `src/harness_recall/cli.py` (add browse command)
**Dependency:** `textual>=0.80`

Layout:
```
┌─ harness-recall ──────────────────────────────────┐
│ Search: █                                          │
│ ┌─ Sessions ──────────┐ ┌─ Preview ──────────────┐│
│ │ ▸ 019cbc4c codex    │ │ User:                  ││
│ │   "Fix auth..."     │ │ Fix the auth bug...    ││
│ │   285b38ce claude   │ │                        ││
│ │   "Design taste.."  │ │ Assistant:             ││
│ └─────────────────────┘ │ I found the bug...     ││
│                         └────────────────────────┘│
│ [↑↓] Navigate [Enter] Full [e] Export [/] Search  │
│ [f] Filter [q] Quit                                │
└───────────────────────────────────────────────────┘
```

Design (warm stone aesthetic matching HTML export):
- Background: stone-900 (#1c1917) for dark TUI
- Accent: teal (#0d9488) for selection/highlights
- Text: stone-100 (#f5f5f4)
- Muted: stone-500 (#78716c)
- No borders with heavy box drawing — use subtle separators
- Session list: role as small caps, snippet preview
- Preview pane: full turn content, tool calls collapsed
- Footer: key bindings bar
